# Куда пойти (Where to Go)

Django-проект с картой интересных мест.  
Локации хранятся в базе, а в админке можно редактировать описания, загружать фотографии и менять их порядок перетаскиванием.

## Запуск

Для запуска нужен Python 3.10+.

1. Установите зависимости:

```sh
pip install -r requirements.txt
```

2. Примените миграции:

```sh
python manage.py migrate
```

3. Создайте администратора:

```sh
python manage.py createsuperuser
```

4. Запустите сервер:

```sh
python manage.py runserver
```

Сайт будет доступен по адресу `http://127.0.0.1:8000/`, админка по адресу `http://127.0.0.1:8000/admin/`.

## Переменные окружения

Часть настроек проекта берётся из переменных окружения через `environs`.

Создайте файл `.env` рядом с `manage.py` и добавьте в него значения по примеру [`.env.example`](.env.example):

- `SECRET_KEY` — секретный ключ Django.
- `DEBUG` — режим отладки (`True` или `False`).
- `ALLOWED_HOSTS` — список разрешённых хостов через запятую.
- `DB_ENGINE` — backend базы данных (по умолчанию `django.db.backends.sqlite3`).
- `DB_NAME` — имя/путь к базе данных (по умолчанию `db.sqlite3`).

## Что есть в проекте

- GeoJSON для карты формируется из БД во `views.py`.
- API деталей локации: `/places/<id>/`.
- Загрузка и раздача картинок через `MEDIA_URL`/`MEDIA_ROOT`.
- В админке:
  - WYSIWYG-редактор (`django-tinymce`) для длинного описания,
  - превью изображений,
  - сортировка картинок drag-and-drop (`django-admin-sortable2`).

## Загрузка данных

Загрузить одну локацию по URL JSON:

```sh
python manage.py load_place "http://адрес/файла.json"
```
